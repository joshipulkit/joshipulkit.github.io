Install Avogardo

October 12, 2023 1-minute read
Linux • FOSS • Ubuntu 22.04 LTS • Avogadrov12
Ubuntu • Jammy • Avogadro

Avogadro is an advanced molecule editor and visualizer designed for cross-platform use in computational chemistry, molecular modeling, bioinformatics, materials science, and related areas. It offers flexible high quality rendering and a powerful plugin architecture. More details about Avogadro can be found here.

Since Ubuntu 20.04 LTS release, the version of Avogadro in its official repos (focal/jammy) has changed from v1.2 to v1.95. However, the seemingly newer version does not offer enough functionality in terms of tweaking bond parameters, UFF optimization, and general ease of use. So, if you are one of those people who crave to use the earlier version of Avogadro (v1.2), this is the place for you.

Follow these simple steps which will guide you through the installation process:

  • Open terminal.
  • Clone this GitHub repository to your desired path
git clone https://github.com/joshipulkit/avogadro-1.2.git
  • Run the installation script after carefully going through the README document
bash install.sh

For any queries regarding this installation, feel free to contact me on email. Enjoy building molecules!
